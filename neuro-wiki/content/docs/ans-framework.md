# Neuro-Fascial ANS Framework v2.5 FINAL

> 근막경선(Anatomy Trains) · 경맥 · 자율신경계를 통합하여  
> 인체를 **입력–변환–분배–조절–출력**의 계층적 시스템으로 해석하는 통합 임상 모델

---

## 핵심 명제

1. **DFL만이 원인이다.** SBL·SFL·LL은 표현이다.
2. **음양은 교감/부교감이 아닌** 구심성(음) / 원심성(양) 방향성이다.
3. **표리는 교감신경절 공유가 아닌** 미주신경 동시지배로 재정의된다.
4. **상한론 육경은** ANS 붕괴의 7단계 순서로 해석된다.

---

## 전체 구조 (Layer Architecture)

### 상지 수직 흐름

```
DFL (코어·입력)
    ↓
횡격막 (ANS 변환 게이트)
    ↓
TJ — 삼초경 (전신 ANS 분배)
    ↓
SFAL (상지 조절: LU · PC)
    ↓
SBAL (상지 출력: LI · SI · HT)
```

### 하지 병렬 출력

```
DFL
├── SBL — 후면 긴장 출력
├── SFL — 전면 압력 출력
└── LL  — 측면 안정 출력
```

---

## 레이어 상세

### 🔵 DFL — Deep Front Line (코어·입력)

**정의:** 지면 반력을 내측 압력으로 변환하여 복압과 척추 안정성을 형성하는 코어 생성·입력 시스템

**해부학적 연속선:**
족저근막 → 후경골근/FDL/FHL → 내전근군 → 골반저근 → 장요근 → 횡격막 → 경부 심부굴근

**신경 특성:** 부교감 기반 / 구심성 우세 / ANS 레벨 1 (중추)

**절대 원칙:**
> DFL만이 원인이다. 나머지는 표현이다.  
> 출력 레이어(SBL·SFL·LL)만 치료하면 반드시 재발한다.

**임상 단서:**
- 복직근 긴장 → 교감↑ / 복직근 무력 → 비경(DFL) 붕괴
- 장요근 단축 → 골반 전방경사 + 요추 과전만
- 후경골근 약화 → 편평족 + 내측 압력 소실

---

### ⚫ 횡격막 — ANS 변환 게이트

**정의:** DFL의 압력 시스템을 자율신경 시스템으로 변환하는 중앙 게이트  
같은 구조에서 **기계(LU) vs 기능(PC)** 이 갈린다.

**신경 분기:**

| 경맥 | 신경 | 기능 | 증상 |
|------|------|------|------|
| LU (Lung) | C3–5 체성신경 (횡격막신경) | 기계적 호흡 | 숨이 안됨 (물리적) |
| PC (Pericardium) | 미주신경 (CN X) | ANS 조절·감정 | 답답함 (기능적) |

**감별 질문:**
- LU → "숨이 실제로 안 쉬어져요"
- PC → "가슴이 답답하고 불안해요"

---

### 🟣 TJ — Triple Jiao (삼초경) 독립 ANS 분배 레이어

**정의:** 특정 경선이나 장기가 아닌, 전신 자율신경 분배 네트워크

| 항목 | 내용 |
|------|------|
| 위치 | 없음 (전신 분포) |
| 장기 | 없음 |
| 신경학적 대응 | 자율신경총 (Autonomic Plexus) |
| ANS 레벨 | 레벨 2 (신경절 불안정) |

**왜 SBAL이 아닌가:**
- 전신 유동 네트워크 → 팔 뒤쪽(SBAL)으로 고정 불가
- SBAL에 포함 시 전신 네트워크를 '팔 뒤쪽 경선'으로 격하시키는 이론적 오류

**TJ 이상 임상 패턴:** 한열왕래 / 전신 불안 / 비정형 통증 (위치 불일정)

---

### 🔵 SFAL — Superficial Front Arm Line (상지 조절)

**LU (요측):** 대흉근 상부 · 소흉근 · 이두근  
**PC (중앙):** 대흉근 하부 · 전거근 · 회내근

> LU와 PC는 연속이 아니라 **분기**이다.  
> 같은 횡격막에서 다른 신경 경로로 갈린다.

---

### 🔴 SBAL — Superficial Back Arm Line (상지 출력)

| 경맥 | 기능 | 핵심 정의 |
|------|------|----------|
| LI (大腸) | 운동 출력 | 파지·조작 |
| SI (小腸) | 구조 지지 | 견갑 안정 |
| HT (心) | 순환·감각·리듬 | 3요소 동시 이상 |

**HT 진단 기준 — 반드시 3요소 동시:**
1. 이상감각 (말초 순환 이상)
2. 냉감
3. 수면 장애 (리듬 이상)

> HT는 원인이 아니라 결과이다. 냉감 단독 = HT 아님.

---

### 🔴 SBL — Superficial Back Line (후면 출력)

**해부학적 연속선:**  
족저근막 → 아킬레스건 → 비복근/가자미근 → 햄스트링 → 척추기립근 → 후두하근

**핵심 기전: Myodural Bridge**  
후두하근 ↔ 경막 직접 연결 → 족저 이상이 긴장성 두통까지 연결

**임상:** 요통 / 긴장성 두통 / 후면 전체 뻣뻣함

---

### 🟡 SFL — Superficial Front Line (전면 출력)

**해부학적 연속선:**  
전경골근 → 대퇴직근 → 복직근 → SCM

**핵심 지표:**  
복직근 상태 = 위장 기능 상태  
- 긴장 → 교감↑  
- 무력 → 비경(DFL) 붕괴

**임상:** 소화 장애 / 복압 저하 / 전면 자세 붕괴

---

### 🟢 LL — Lateral Line (측면 안정)

**구성:** 비골근 → TFL → IT band → 중둔근 → 외복사근/내복사근

**핵심 구조:** Y (하지) → X (체간) 구조  
**임상:** IT band syndrome / 골반 회전 / 측면 불안정

---

## 음양 재정의

**기존:** 음 = 부교감 / 양 = 교감  
**재정의:** 음 = 구심성(afferent) / 양 = 원심성(efferent)

| | 음 (陰) | 양 (陽) |
|---|---|---|
| 방향성 | 구심성 (중추로 향함) | 원심성 (말초로 향함) |
| 기능 | 감각·수렴·입력 | 운동·발산·출력 |
| 경맥 | 족삼음·수삼음 | 족삼양·수삼양 |

---

## 표리 재정의

**기존:** 교감신경절 공유 기준  
**재정의:** 미주신경 동시지배 기준

**예시:**  
폐수(T3) ≠ 대장수(T11) — 교감신경절 레벨 전혀 다름  
그러나 **LU·LI 표리 성립** = 미주신경이 폐·대장 동시지배

| 표리 쌍 | 기전 | 미주신경 지배 |
|---------|------|-------------|
| LU ↔ LI | 호흡기·대장 동시 조절 | ✓ |
| ST ↔ SP | 위장 기능 동시 조절 | ✓ |
| HT ↔ SI | 심장·소장 동시 조절 | ✓ |

## 상한론 ANS 붕괴 7단계

| 단계 | 육경 | ANS 상태 | 핵심 처방 |
|------|------|----------|----------|
| 1 | 太陽 (태양) | 체표 교감↑, 피부 방어 | 桂枝湯·麻黃湯 |
| 2 | 少陽 (소양) | TJ 불안정, 한열왕래 | 小柴胡湯 |
| 3 | 陽明 (양명) | 교감 항진 극대, 열·조 | 白虎湯·承氣湯 |
| 4 | 太陰 (태음) | 부교감 기능 저하, 복냉 | 理中湯·四逆湯 |
| 5 | 少陰 (소음) | 심신 ANS 붕괴, 맥침 | 麻黃附子細辛湯 |
| 6 | 厥陰 (궐음) | 교감·부교감 혼란, 상열하한 | 烏梅丸 |
| 7 | 崩壞 (붕괴) | 완전 ANS 실조 | 回陽救逆 |

## 임상 진단 알고리즘

```
1단계: DFL 평가
   복압 있나? → 장요근·골반저 촉진
   족저 내측 압력 있나?

2단계: 횡격막 게이트 분기 확인
   LU (물리적 호흡 제한)?
   PC (ANS·감정 문제)?

3단계: TJ 레벨 확인
   한열왕래 있나?
   비정형 전신 증상?

4단계: SFAL/SBAL 출력 평가
   LI (운동) / SI (구조) / HT (3요소 동시)?

5단계: 하지 출력 평가
   SBL (후면) / SFL (전면) / LL (측면)

→ 항상 DFL로 돌아와서 치료
```
